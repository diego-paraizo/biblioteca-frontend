# Library Manager - Frontend

Este é o módulo de interface do sistema de Gerenciamento de Biblioteca, desenvolvido com as funcionalidades mais recentes do Angular. A aplicação foca em uma experiência de usuário reativa para a gestão de acervos.

## 🚀 Tecnologias Utilizadas
* **Angular 21** (Standalone Components & Signals)
* **Bootstrap 5** (Sistema de Grid e Componentes UI)
* **TypeScript**
* **NPM** (Gerenciador de Dependências)

## 🛠️ Funcionalidades Implementadas
* **Formulários Reativos:** Cadastro de livros com validações de tipos e limites numéricos.
* **Máscaras de Input:** Tratamento dinâmico para campos de preço (R$) e edição.
* **Gerenciamento de Estado:** Uso de Signals para fluxo de dados performático.
* **Consumo de API:** Service layer estruturada para integração com o Backend.

## 🏁 Como rodar na máquina local

### Pré-requisitos
* **Node.js** (Versão 20.x ou superior recomendada)
* **Angular CLI** (`npm install -g @angular/cli`)

### Instruções
1. **Instalação:**
   ```bash
   npm install

2. **Execução:**
   ```bash

3. **Acesso:**
